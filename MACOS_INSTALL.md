# macOS 安装说明

## 安全警告说明

由于应用未经过 Apple 代码签名，macOS 的 Gatekeeper 会显示安全警告。这是正常的安全机制。

## 安装方法

### 方法 1：右键打开（推荐）

1. **不要直接双击 DMG 文件**
2. **右键点击** `JSON-Workbench-0.1.7.dmg` 文件
3. 选择 **"打开"**
4. 在弹出的警告对话框中，点击 **"打开"**
5. 将应用拖拽到 Applications 文件夹

### 方法 2：系统设置允许

1. 双击 DMG 文件（会显示警告）
2. 打开 **"系统设置"** → **"隐私与安全性"**
3. 在 **"安全性"** 部分，找到被阻止的应用
4. 点击 **"仍要打开"** 按钮

### 方法 3：终端命令（高级用户）

如果上述方法都不行，可以在终端中运行：

```bash
# 移除隔离属性
xattr -cr /path/to/JSON\ Workbench.app

# 然后就可以正常打开了
open /path/to/JSON\ Workbench.app
```

## 为什么会出现这个警告？

- macOS Gatekeeper 会检查应用的代码签名
- 未签名的应用会显示此警告
- 这是 macOS 的正常安全机制，不是病毒或恶意软件

## 未来改进

我们计划在未来版本中添加 Apple 代码签名，以消除此警告。
